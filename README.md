# Teste Blog Mobile Friendly

Blog simples, responsivo e feito com HTML, CSS e JS puro.

## Objetivo
Criar uma base leve para estudo, portfólio e evolução do projeto.

## Recursos
- layout mobile first
- menu responsivo
- busca local de posts
- botão de curtir
- estrutura simples
- sem dependências externas

## Estrutura
teste-blog-mobile-friendly/
├── README.md
├── index.html
├── style.css
└── script.js

## Como rodar
1. Clone o repositório.
2. Abra o arquivo `index.html` no navegador.
3. Edite o conteúdo no HTML.
4. Ajuste o visual no CSS.
5. Atualize as interações no JS.

## Linha do tempo
1. README inicial
2. Base HTML do site
3. Estilo responsivo
4. Interações em JavaScript
5. README final

## Histórico de commits
- docs: inicia readme
- feat: cria base do site
- style: ajusta layout mobile
- feat: ativa menu e busca
- docs: finaliza readme

## Resultado
Um blog estático simples, leve e pronto para servir como base de estudo ou portfólio.
